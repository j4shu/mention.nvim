# mention.nvim

## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues (`j4shu/mention.nvim`) via the `gh` CLI. See `.claude/docs/agents/issue-tracker.md`.

### Triage labels

Default label vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `.claude/docs/agents/triage-labels.md`.

### Domain docs

Single-context: `CONTEXT.md` and `.claude/docs/adr/` at the repo root. See `.claude/docs/agents/domain.md`.
